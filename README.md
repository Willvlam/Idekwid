## Impossible 2D Game

A super difficult 2D platformer game built with HTML, CSS, and JavaScript.

### Features
- Multiple levels with increasing difficulty (up to infinite)
- Platforming mechanics with high precision required
- Enemy avoidance with fast-moving enemies
- Puzzle elements with moving platforms and invisible traps
- Time limits per level
- Lives system: 3 lives, game over at 0 lives

### How to Play
- Use WASD or arrow keys to move and jump
- Avoid enemies and traps
- Collect yellow items to advance levels
- Press P to pause

### Running the Game
1. Start a local HTTP server:
   ```bash
   python3 -m http.server 8000
   ```
2. Open your browser and go to `http://localhost:8000`

### Game Mechanics
- Gravity and friction for realistic movement
- Collision detection with platforms, enemies, and collectibles
- Lives system: lose a life on death, game over at 0 lives
- Score increases by collecting items
